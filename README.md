# Chao Chien Wang｜資料分析與商業分析

我是一名具備製造業與財務風險經驗的資料／商業分析工作者，擅長將複雜的營運流程轉化為可計算、可監控、可自動化的資料產品。

目前專注於製造、供應鏈與風險管理場景，從需求釐清、資料整合、業務規則設計，到儀表板、預警與流程上線，參與完整的分析專案生命週期。

## 我能解決的問題

- 整合分散於不同系統的資料，建立一致的決策依據
- 將業務經驗轉化為可重複執行的計算與判斷規則
- 透過儀表板與主動預警，縮短從發現問題到採取行動的時間
- 協調業務與技術團隊，讓分析成果真正進入日常作業流程

## 精選專案

### 1. [外匯風險自動拋轉系統](https://github.com/ChienChienChien/FX_Hedging_Automation)

整合銷售、採購、發票與匯率資料，將合約成立、金額調整、取消、立帳結清等事件轉換為一致的外匯風險部位與交易資料，自動銜接部位及損益管理流程。

- 每月處理約 1.5 億美元之外匯風險
- 整合生產、財務、採購與部位分析系統
- 將跨部門風險認定與匯率規則轉化為自動化流程
- 負責需求釐清、業務規則設計、資料處理與跨部門協作

### 2. [原料成本控制與庫存預測系統](https://github.com/ChienChienChien/Material_Forecasting_System)

整合原料庫存、採購、進貨、生產計畫與 BOM 資料，建立未來三個月的每日庫存推估與多級風險預警，協助管理者提前處理可能的斷料風險。

- 管理超過 50 種原料及每月約新台幣 10 億元的成本規模
- 建立未來三個月的庫存推估與每日更新流程
- 透過 Power BI 呈現庫存趨勢、風險原料與明細
- 每日自動將分級警示推送至協作平台

### 3. [BOM 智慧運算與管理平台](https://github.com/ChienChienChien/BOM_Management_Platform)

將 BOM 參數維護、資料驗證、運算、版本管理、發布與現場觸發整合成可管理的作業流程，支援原料耗用規劃與生產決策。

- 建立參數維護、簽核、驗證及發布流程
- 串接 Python、資料庫、Microsoft 365 與製造執行系統
- 支援排程運算及現場即時觸發
- 將原本分散的作業整合為可追蹤、可維護的流程

### 4. 製造資料品質監控平台

針對製造資料管線建立欄位、資料表與更新日期等品質規則，讓資料異常能在影響報表與決策前被發現。

- 建立可設定的資料品質檢查規則
- 整合資料轉檔、資料庫寫入與品質檢查流程
- 透過儀表板集中呈現資料品質狀態
